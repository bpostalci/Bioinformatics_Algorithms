# General Information
##### usage: bwtfm [option]
#####        Options:
#####        --help                                      :  display help
#####        --index  [text].fa                          :  outputs [text].fa.bwt and [text].fa.fm index files
#####        --search [text].fa --pattern [query].fa     :  file to be searched by a specified pattern file (both text and pattern files must be supplied)
